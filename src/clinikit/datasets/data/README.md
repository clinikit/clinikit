# Bundled UCI benchmark datasets

The CSV files in this directory are public-domain or
research-redistributable copies of well-known UCI Machine Learning
Repository benchmarks. They are shipped inside the `clinikit`
package so the loaders work after a clean `pip install clinikit`
with no network access.

The files are generated by [`scripts/fetch_datasets.py`](../../../../scripts/fetch_datasets.py)
and committed to the repository — refresh the directory by re-running
that script.

| File           | Source                                                          | Samples | Features | License / status                                |
| -------------- | --------------------------------------------------------------- | ------- | -------- | ----------------------------------------------- |
| `pima.csv`     | UCI / NIDDK PIMA Indians Diabetes                               |     768 |        8 | Public domain (US Government collection).        |
| `wisconsin.csv`| UCI Wisconsin Breast Cancer (Diagnostic), via `sklearn.datasets` |     569 |       30 | UCI redistributable for research.               |
| `heart.csv`    | UCI Heart Disease (Cleveland subset, processed)                  |     303 |       13 | UCI redistributable for research.               |

See [`../../../../CITATIONS.md`](../../../../CITATIONS.md) for the
academic references behind each dataset.
