# clinikit documentation

Prepared by Berat Kaan SEVEN.

A lightweight, sklearn-compatible Python toolkit for tabular machine
learning. `clinikit` bundles 14 hybrid classifiers, calibration
utilities, label-noise diagnostics, fairness audits, experiment
protocols, and structured HTML reports behind a single drop-in package.

> Research and development use only. `clinikit` is an integration
> toolkit, not a regulated product and not a research paper of
> original methods.
